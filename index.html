"chunes"

"chunes" is a music notation sharing site for folk, traditional, and original written music, using abc notation, and React component called react-abc. Transcribed music can be shared to the public collection, where it can be read, discussed, and even listened to! User can also fork other authors' work, and edit it inside of your personal library. I built my front-end in React, and then decided to use Django to serve up the database from the back-end. All of the information I could find online about this specific set-up also included bringing in more libararies, which I didnt want to do before I understood the framework more thoroughly.
